# i18n 中英文双语切换 (2026-06-26)

**Why:** App 所有 UI 硬编码中文，国际化为零。需要中英文切换以覆盖更广用户群。

**How to apply:** AppBar 右上角有语言切换按钮（中/EN），点击即时切换所有 UI 文字。

## 实现方案
- `lib/i18n/strings.dart` — `Strings` 类包含所有翻译，通过 `InheritedWidget` 树传播
- `StringsScope` StatefulWidget 包裹根 App，`Strings.of(context)` 获取实例
- `Strings.setLocale(context, locale)` 切换语言，触发全树重建
- `Strings.currentLocale` 静态字段供非 Widget 代码（如 InsightService）使用

## 涉及文件
- `lib/i18n/strings.dart` — 新建：~220 行翻译键值
- `lib/main.dart` — 修改：StringsScope 包裹 + 语言切换按钮 + 所有 UI 字符串
- `lib/ui/search_dialog.dart` — 修改：搜索相关字符串翻译
- `lib/ui/compare_page.dart` — 修改：对比页翻译
- `lib/ui/info_page.dart` — 修改：科普页全文翻译（含英文 bullet parsing）
- `lib/services/insight_service.dart` — 修改：洞察文本双语生成
